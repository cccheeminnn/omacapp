Welcome to the Help! page.

The app works pretty straightforward, user should upload a .csv file with populated with street names like the following files. 50, 250, or 500 Singapore street names

The two things to look out for are the column header should be "address" and the column fields should ONLY consist of the street name.

Depending on how many streets are uploaded the query would take longer/shorter.

After the query is complete, a file is created on DO spaces and a direct download link to the file is given to the user,
the user can then either choose to download to the file or have it sent to their email instead!
