Welcome to the Help! page.

The app works pretty straightforward, user should upload a .csv file populated with street names like the following files. 10, 15, or 25 Singapore street names

As Heroku has a request timeout duration of 30s, for 20 and above searches we recommend having the file sent straight to your email instead.
The two things to look out for are the column header should be "address" and the column fields should ONLY consist of the street name.

After the query is complete, a file is created on DO spaces and a direct download link to the file is given to the user, the user can then either choose to download the file locally or have it sent to their email instead.
